# Tailwind CSS für das Ivy Child Theme

## Entwicklung

1. Stelle sicher, dass Node.js und npm installiert sind.
2. Wechsle ins Theme-Verzeichnis:
   ```
   cd /Users/rainer/_OWN/ivy/ivy
   ```
3. Installiere die Abhängigkeiten:
   ```
   npm install
   ```
4. Starte den Entwicklungsmodus (automatisches Bauen bei Änderungen):
   ```
   npm run dev
   ```
   Oder für einmaligen Build:
   ```
   npm run build
   ```

## Hinweise
- Passe deine Tailwind-Klassen in den PHP-Templates an.
- Die generierte CSS-Datei liegt in `/dist/style.css` und wird automatisch eingebunden.
- Die Datei `style.css` dient nur als Theme-Header für WordPress.
