# Ivy Child Theme

Dies ist ein Child Theme für das Storefront WordPress Theme.

## Installation

1. Kopiere den Ordner `ivy` in dein WordPress `wp-content/themes` Verzeichnis.
2. Aktiviere das Theme im WordPress Backend unter Design > Themes.

## Hinweise
- Das Parent Theme "Storefront" muss installiert sein.
- Eigene Anpassungen können in `style.css` und `functions.php` vorgenommen werden.
